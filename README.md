– Proyectos Java (Colecciones y Estructuras)


Este repositorio reúne una serie de proyectos desarrollados en Java con el objetivo de aplicar el Framework de Colecciones de Java y reforzar conceptos clave de programación orientada a objetos, estructuras de datos y manipulación de archivos.

A lo largo de los ejercicios se implementaron diferentes sistemas de consola que simulan problemas reales, utilizando estructuras como ArrayList, HashSet, HashMap, Stack, Queue y LinkedList.

El propósito principal fue comprender cómo funcionan las colecciones en Java y cuándo usar cada una según el tipo de problema.

🎯 Objetivos del trabajo
Aplicar estructuras de datos del Java Collections Framework.
Practicar programación orientada a objetos.
Implementar menús interactivos por consola.
Aplicar ordenamientos con Comparator.
Manipular archivos para persistencia de datos.
Desarrollar lógica de negocio separada de la interfaz.
🧠 Tecnologías utilizadas
Java JDK 21
Java Collections Framework
Programación Orientada a Objetos (POO)
Manejo de archivos (java.io)
Consola interactiva (Scanner)
